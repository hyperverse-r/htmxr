# Package index

## Page

Build a complete HTML page with htmx support.

- [`hx_page()`](https://hyperverse-r.github.io/htmxr/reference/hx_page.md)
  : Generate a complete HTML page with htmx
- [`hx_head()`](https://hyperverse-r.github.io/htmxr/reference/hx_head.md)
  : Specify additional head elements for an htmxr page
- [`hx_serve_assets()`](https://hyperverse-r.github.io/htmxr/reference/hx_serve_assets.md)
  : Serve htmxr static assets

## Inputs

Interactive input components wired to htmx requests.

- [`hx_button()`](https://hyperverse-r.github.io/htmxr/reference/hx_button.md)
  : Button element
- [`hx_select_input()`](https://hyperverse-r.github.io/htmxr/reference/hx_select_input.md)
  : Select input
- [`hx_slider_input()`](https://hyperverse-r.github.io/htmxr/reference/hx_slider_input.md)
  : Slider input

## Table

Generate and update HTML tables via htmx swaps.

- [`hx_table()`](https://hyperverse-r.github.io/htmxr/reference/hx_table.md)
  : Table with htmx-powered tbody
- [`hx_table_rows()`](https://hyperverse-r.github.io/htmxr/reference/hx_table_rows.md)
  : Table rows fragment

## htmx modifiers

Add or modify htmx attributes on any HTML element.

- [`hx_set()`](https://hyperverse-r.github.io/htmxr/reference/hx_set.md)
  : Add htmx attributes to any HTML tag

## Server

Server-side utilities for handling htmx requests.

- [`htmxr_is_htmx()`](https://hyperverse-r.github.io/htmxr/reference/htmxr_is_htmx.md)
  : Detect if a request comes from htmx

## Developer tools

Helpers for running and exploring built-in examples.

- [`hx_run_example()`](https://hyperverse-r.github.io/htmxr/reference/hx_run_example.md)
  : Run an htmxr example
